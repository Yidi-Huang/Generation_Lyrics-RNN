Lyric Generator

Generate lyrics in your favourite style

1. Les objectifs du projet

L’objectif du projet est de créer automatiquement des paroles de chansons basées sur les mots dans le style de différents chanteurs, en utilisant un Réseau de Neurones Récurrents (RNN), à partir d'une première phrase donnée.

2. Les données utilisées

Les sources utilisées pour ce projet comprennent :

  • 11 ensembles de données, chacun contenant 50 chansons d’un chanteur. Les données sont disponibles à l'adresse suivante : https://github.com/Yidi-Huang/Generation_Lyrics-RNN/tree/main/corpus
  • Format : fichier CSV avec deux colonnes : "Artiste" et "Texte".
  • Origine : récupéré depuis le site https://genius.com/
  • Statut juridique : conforme aux conditions d'utilisation du site Genius.com.

Le projet s’inspire du notebook pour la génération des paroles de chansons basés sur les caractères disponible sur Kaggle.

3. La méthodologie

Le travail est divisé en 5 étapes :

  1. Récupération des données : Les données sont récupérées à partir des sources mentionnées précédemment.
  2. Construction du modèle RNN : Un modèle RNN est construit pour générer les paroles de chansons avec plusieurs ajustements spécifiques pour résoudre certains problèmes rencontrés.
  3. Fine-tuning et test : Pour améliorer la performance du modèle, un fine-tuning est effectué.
  4. Mise en pratique sur la page web à l’aide de FastAPI : Le modèle est mis en œuvre sur une page web.
  5. Mise en forme des pages avec la documentation : Les pages web sont mises en forme pour que la documentation soit claire et accessible aux utilisateurs.

4. L’implémentation

Différents outils et technologies ont été utilisés dans ce projet :

  • LyricsGenius : pour Genius.com API
  • LSTM : modèle de base
  • Keras : API pour les réseaux de neurones

5. Les résultats et discussion

Des améliorations futures sont envisagées pour rendre les paroles plus naturelles, telles que la notification des rimes et la création des paroles selon les thèmes ou les émotions.
